# Product Overview

FlutterToNative.pro is a premium educational platform helping senior Flutter developers transition to native iOS development.

## Core Value Proposition

- Target audience: Senior Flutter engineers with existing mobile development experience
- Focus: High-leverage content that respects developer expertise
- Business model: Freemium with 20% free content, premium gating for advanced topics

## Key Features

- **Interactive Widget Map**: Side-by-side Flutter ↔ SwiftUI comparisons
- **Architecture Guides**: MVVM, Clean Architecture, SPM Modularization
- **Interview Prep**: 50+ curated iOS interview questions with premium gating
- **Lead Magnet**: "Top 10 Performance Pitfalls" PDF for email capture
- **Progress Tracking**: Cloud-synced course progress via Supabase
- **Premium Subscriptions**: Stripe-powered payment system

## Content Strategy

- Mental model mapping (Flutter concepts → Swift/iOS equivalents)
- Professional architecture patterns for iOS
- Interview preparation focused on senior-level questions
- Practical implementation examples (e.g., expense tracker app)

## Premium Gating Rules

- Architecture: Only MVVM is free, rest requires premium
- Interview Prep: First 3 topics free (ARC, Value vs Reference, Concurrency)
- Visual indicators: Lock icons on premium content in sidebar
